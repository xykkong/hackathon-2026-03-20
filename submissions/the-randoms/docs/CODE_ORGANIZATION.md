# Code Organization

## Source of Truth

All runnable project code is under:

- `src/react-video-client-avatar` (frontend)
- `src/simple-backend` (demo backend)
- `src/server-custom-llm` (advanced/custom backend variants)

## Non-Code Folders

- `assets/`: screenshots and media assets
- `docs/`: project and engineering documentation
- `design/`: design references and working notes

## Startup Order (Demo)

1. Start `src/simple-backend`
2. Start `src/react-video-client-avatar`
3. Open the frontend and start a call

## Notes

- `demo.txt` is kept at repository root and contains the demo video link.
- Main setup instructions are maintained in `README.md`.
- Final submission checks are tracked in `docs/SUBMISSION_CHECKLIST.md`.
